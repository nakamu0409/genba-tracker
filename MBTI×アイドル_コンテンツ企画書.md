# MBTI × アイドル コンテンツ企画書

## 1. コンセプト

MBTI診断とアイドルファン心理を組み合わせたショート動画/画像コンテンツ。
特定の実在アイドルではなく「あなたの推し」「〇〇系アイドル」という抽象的な表現を使うことで、権利問題を避けつつ広い視聴者に当てはめてもらう設計。

## 2. 投稿フォーマット

### 形式: 画像カルーセル(TikTok複数画像投稿 / Instagramカルーセル)

```
スライド1: タイトル(フック)
スライド2: 特徴①
スライド3: 特徴②
スライド4: 特徴③
スライド5: まとめ・問いかけ(コメント誘導)
スライド6(任意): CTA・次回予告
```

- 各スライドは1メッセージに絞る(情報を詰め込みすぎない)
- フォントは太め・大きめで1〜2種類に統一
- タイプごとに背景色を分けるとシリーズ感が出てフォロワーに覚えてもらいやすい
  - 例: 直観(N)系 = 紫、感覚(S)系 = 緑 など

## 5. 運用上の注意点

- 実在アイドルの名前・画像は直接使用しない(肖像権・著作権リスク回避)
- 「〇〇系アイドル」という抽象表現に統一し、視聴者自身に当てはめてもらう設計を維持する
- 投稿頻度は週3〜5本を目安に開始し、データを見て調整する

## 6. Canva作成手順

### 6.1 新規デザインの作り方

1. Canvaにログイン後、右上の「デザインを作成」をクリック
2. 検索欄に「カスタムサイズ」と入力し、**1080 × 1350px**(TikTok/Instagram両対応の縦長比率)を指定して作成
3. 1枚作ったら「ページを複製」して同じデザインのまま量産する(スライド5枚なら5回複製)
4. 完成後は「共有」→「ダウンロード」→ファイルの種類を「JPG」または「PNG」、「すべてのページ」を選択して書き出す

### 6.2 共通デザインルール

| 項目 | 指定 |
|---|---|
| キャンバスサイズ | 1080 × 1350px(縦長) |
| フォント | 見出し: 太字のゴシック系(例: Poppins Bold / もしくは日本語の「ふい字」「M PLUS Rounded 1c Bold」) |
| 見出し文字サイズ | 60〜80pt |
| 本文文字サイズ | 36〜48pt |
| 文字色 | 背景が濃色の場合は白、薄色の場合は黒 |
| 余白 | 上下左右に120px程度のマージンを確保し、文字を中央寄せ |

### 6.3 タイプ別カラー指定

| 軸 | タイプ例 | 背景色(Canvaのカラーコード) |
|---|---|---|
| 直観(N)系 | INFJ, ENFP, INTJ, ENTP など | `#7C5CFC`(紫) |
| 感覚(S)系 | ISTJ, ESFP, ISFJ, ESTP など | `#4CAF7D`(緑) |
| 思考(T)系のアクセント | - | `#2D2D2D`(濃グレー、文字や枠線に使用) |
| 感情(F)系のアクセント | - | `#FF6F91`(ピンク、文字や枠線に使用) |

### 6.4 サンプル①「ENFP系アイドルの特徴」スライド別指定

| スライド | 背景色 | テキスト配置 | テキスト内容 |
|---|---|---|---|

| 2 | 紫 `#7C5CFC` | 上部に番号、中央に本文 | ①感情表現が豊か / 嬉しい時も悲しい時も全身で伝えてくれる |

| 4 | 紫 `#7C5CFC` | 同上 | ③アドリブの天才 / 台本通りじゃない瞬間こそ一番輝く |
| 5 | 紫 `#7C5CFC`(やや濃いめのグラデーション推奨) | 中央、問いかけ調 | あなたの推し、当てはまった?コメントで教えて |

※ サンプル②(INFJ)も同じ紫系、サンプル③(MBTI別診断)は4タイプ混在のため、スライドごとに紫/緑を切り替えるか、グレー系で統一すると見やすくなります。

### 6.5 作業効率化のポイント

- 1枚目を完成させたら「このページのスタイルを他のページにも適用」または手動複製で量産すると時間短縮になる
- Canvaの「ブランドキット」機能(無料プランでも一部利用可)にフォント・カラーを登録しておくと、毎回の設定が不要になる
- 16タイプ分を量産する場合は、タイプごとに「紫(N系)」「緑(S系)」のテンプレートを1つずつ作り、テキストだけ差し替える運用が最も早い
